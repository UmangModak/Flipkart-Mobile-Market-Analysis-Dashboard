# Flipkart Mobile Market Analysis Dashboard

# Project Overview

This project analyzes the Flipkart mobile dataset using Power BI to understand pricing trends, brand positioning, and customer ratings in the Indian smartphone market. The dashboard provides interactive insights into selling prices, rating distribution, and brand-level comparisons.

# Key Analysis

The dashboard evaluates multiple metrics including maximum selling price, minimum selling price, average selling price, and rating performance across different mobile brands such as Apple, Samsung, Google Pixel, ASUS, and others.

Key insights include:

* Apple and Google Pixel appear in the premium price segment with the highest average selling prices.
* Samsung, Vivo, and HTC operate across mid-range price categories.
* Brands like Gionee and Nokia appear in the lower price segments.

The analysis also explores rating distributions across brands to understand product popularity and perceived quality.

# Dashboard Features

* Brand level price comparison (Max, Min, Average selling price)
* Rating distribution analysis by brand
* Interactive filters for brand, model, and selling price range
* KPI indicators showing dataset coverage and counts

# Tools Used

* Power BI
* Power Query
* Data Modeling
* DAX Measures

# Outcome

The dashboard helps identify price positioning strategies across brands and provides insights into how different smartphone manufacturers compete within various price segments in the Indian market.
